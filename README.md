# 앨범 게시판 - 개인프로젝트

## 개발환경

Front-End

* HTML5/CSS3/JAVASCRIPT
* jQuery / AJax
* BootStrap Framework

Back-End

* JAVA
* JSP/Servlet
* Spring Framework
* Mybatis
* Tomcat

Data-Base
* oracle
